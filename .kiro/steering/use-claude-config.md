# Canonical rules location

All project rules and workflows are maintained in:
1. **User-level**: `~/.claude/CLAUDE.md` and `~/.claude/rules/` (universal)
2. **Project-level**: `.claude/CLAUDE.md` and `.claude/rules/` (project-specific)  
3. **Skills**: `.claude/skills/` and `~/.claude/skills/`

Read and follow those files. This directory is maintained for Kiro compatibility only.
